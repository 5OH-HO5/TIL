### Branch

> 브랜치를 배워봅시다.

1. 브랜치 생성: `git branch 브랜치이름`
2. 브랜치 확인: `git branch`
3. 브랜치 이동:  `git checkout 브랜치이름`
4. 브랜치 삭제:  `git branch -d 브랜치이름`
5. 브랜치 생성 + 이동: `git checkout -b 브랜치이름`
